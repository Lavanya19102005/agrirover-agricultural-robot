## Description
AGRIROVER is an autonomous agricultural robot designed to automate key floriculture tasks 
such as seed hole preparation, seed dispensing, and hole closing. The system reduces manual 
labor and improves efficiency through sensor-based automation.

## Features
- Automated seed hole preparation and dispensing
- Soil pH and moisture monitoring
- GPS-based plant location tracking
- Reduced dependency on manual labor

## Technologies Used
- Arduino
- Embedded C
- pH Sensor
- Moisture Sensor
- GPS Module
- Motor Drivers

## Applications
- Floriculture automation
- Smart agriculture
- Sustainable farming solutions
